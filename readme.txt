== Description ==
Frontend Dashboard Captcha WordPress plugin is a supportive plugin for Frontend Dashboard to protect against spam in Login and Register form.

=Login=

Stop Brute force attack on login page using the Captcha

=Register=

Stop botnet registration on register page using the Captcha

=Google Captcha=

Uses Google captcha through their API service.

== Installation ==
1. Upload the “frontend-dashboard-captcha” directory to the plugins directory.
2. Go to the plugins setting page and activate “Frontend Dashboard Captcha”
3. Go to Frontend Dashboard | Captcha and then add the Google Captcha keys
4. Do save.
